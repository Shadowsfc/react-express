# React Express App

An react - express example application for downloding files using express api.

For more details, see https://flaviocopes.com/express-send-files/.


## Installation

    git clone https://github.com/skdarekar/react-express.git
    cd react-express
    cd backend-express
    npm install
    npm start
    cd ..
    cd frontend
    npm install
    npm start
